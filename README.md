# Google Auth React

A simple authentication project with React, TypeScript and Firebase.

## Installation

### Clone

```
git clone 
```

### Access the folder

```
cd <name-folder>
```

### Install dependencies

```
npm install
```

### Start the app

```
npm run dev
```

### Port:

```
http://localhost:5173/
```
